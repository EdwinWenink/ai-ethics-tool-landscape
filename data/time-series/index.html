Alibi Detect

Alibi Detect is an open source Python library (sister library to Alibi ) focused detecting outliers, adversial examples, and concept drift. Finding adversial examples is relevant for assessing the security of machine learning models. Machine learning models learn complex statistical patterns in datasets. If these statistical patterns “drift” (in unforeseen ways) after a model is deployed, this will decrease the model performance over time. In systems where model predictions have an impact on people, this may be a threat to the fairness of the predictions. Read more...
